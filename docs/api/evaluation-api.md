# Evaluation API

The evaluation APIs cover Phase 2 (Technical) and Phase 3 (Presentation) scoring.

## Technical Score
`POST /api/v1/technical-score`

See [Scoring API](scoring-api.md) for details.

## Presentation Score

See [Scoring API](scoring-api.md) for details.
